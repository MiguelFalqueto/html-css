Hierarquía de títulos

Como faço essa hierarquía?

Para fazer isso, você usa o mesmo código
do título, porém, aumentando o número que vem depois.

Exemplos:

Título principal = h1.
Título secundário = h2.
E assim por diante até o h6 que é o limite.
